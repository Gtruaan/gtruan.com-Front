# gtruan.com Front-end

Built with React on top of Vite and vanilla CSS.
Site is running (or should be) on [gtruan.com](https://gtruan.com).

Sprites were made by me on **paint.net** :smile:.

## Development

Installation

```bash
yarn install
```

---------------

Running

```bash
yarn start
```

---------------

Building

```bash
yarn build
```
